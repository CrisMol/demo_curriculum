Sistema Hilda María

Procesos Automatizados.

Sistema de venta, alquiler, reportes y todo lo necesario para el buen funcionamiento del negocio, esto permitio un mejor control de empleados, gestión de productos, bodega, entre otros.


Descripción de la Empresa:

Nombre de la Empresa: Mega Bahía

Ubicación: Selva Alegre y Gaspar de Carvajal.

Descripción:
Venta de productos tecnológicos y para el hogar.


Aspectos Técnicos:

Herramientas: PHP 7, Javascript, CSS3, HTML5, MySql.

Cargo: Diseño Web y Módulo "Gestión de Empleados".

Funciones:

Diseño Web: Interfaz y pantallas adaptables a cualquier pantalla de dispositivo del sistema.

Módulo de Empleados: Resgistro y verificación de empleados del negocio hacia las diferentes módulos del sistema.

Descripción:
Proyecto realizado en la Universidad para un Negocio de Alquiler en la que estuve a cargo del diseño web del sistema, este haría todo el proceso de venta, alquiler, reportes, stock y demás características del negocio.
