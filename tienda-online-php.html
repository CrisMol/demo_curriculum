Desarrollo de una Tienda Online de Camisetas.

Proyecto que simula una tienda online realizado con PHP 7, MVC, POO y Mysql.

Utilizando fundamentos de PHP 7 MVC, POO y de bases de datos, se desarrollo una tienda desde cero hasta un nivel intermedio, se realizó respetando los fundamentos y las buenas prácticas de la Programación orientada a objetos y Modelo-Vista-Controlador. Validaciones de urls, usuarios, stock, ventas, imagenes, etc.


Características:

Herramientas y conocimientos aplicados: PHP 7, MVC, POO, Sesiones, Almacenamiento y control de imagenes con Base de datos MySql, HTML, CSS.

Funcionamiento:

  • Registro de Usuarios: Registro de usuarios al portal para realizar pedidos.

  • Registro y gestión de pedidos: Urls validadas para el acceso y control correspondiente de cada usuario, cambiar estado y confirmar pedidos.

  • Registro y gestión de categorías: Creación, actualización y eliminación de categorías de productos.

  • Validación de Usuarios y Control de acceso: Validación de urls, control de acceso a las determinadas pantallas según el rol y login del usuario.

  • Control de productos: Creación, actualización y eliminación de productos e información como stock.

  • Botón agregar al Carrito: Funcion de agregar al carrito y vaciar carrito, mediante sesiones, validaciones de login, sentencias sql en la base de datos.

  • Detalle del pedido: Descripción de cada pedido, mostrando información de productos, total, unidades, dirección de envió y datos del usuario.

  • Código: Tienda de camisetas online GitHub.
