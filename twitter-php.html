Desarrollo de una Pequeña Red Social Parecida a Twitter

Proyecto que simula una pequeña red social con Laravel 7 y Mysql.

Utilizando el framework Laravel y su versión 7 con bases de datos, se desarrollo una mini red social parecida a Twitter desde cero hasta un nivel intermedio. Validaciones de usuarios, likes, comentarios, registros, imagenes, etc.


Características:

Herramientas y conocimientos aplicados: Laravel 7, Base de datos MySql, Ajax, HTML, CSS.

Funcionamiento:

  • Registro de Usuarios: Registro de usuarios nuevos con un nikname, avatar.

  • Comentarios: Registro de comentarios en cada una de las imagénes subidas por los ditintos usuarios.

  • Likes: Likes de cada usuario en forma de corazón que se pone rojo cuando se da like a una imagen.

  • Subida de imágenes: Subida de imágenes de cada usuario con una descripción, este aparece en el perfil de cada uno.

  • Favoritos: Imágenes favoritas de cada usuario.

  • Código: Red Social Twitter GitHub.
