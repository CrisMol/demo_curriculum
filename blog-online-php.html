Desarrollo de un Blog para Videojuegos

Proyecto que simula un blog realizado con PHP 7 y Mysql.

Utilizando fundamentos de PHP 7 y de bases de datos, se desarrollo un blog desde cero hasta un nivel intermedio. Validaciones de usuarios en temas, categorías, login, registro, entradas, etc.


Características:

Herramientas y conocimientos aplicados: PHP 7, Sesiones, Base de datos MySql, HTML, CSS.

Funcionamiento:

  • Registro de Usuarios: Registro de usuarios nuevos con un rol determinador para ingresar al sistema y poder realizar las funciones correspondientes.

  • Validación de Usuarios: Usuarios con acceso validado para poder crear, eliminar o editar nuevas entradas o categorías según corresponda su rol en el sistema.

  • Validación de páginas: Urls validadas para el acceso y control correspondiente de cada usuario.

  • Base de datos: Conexión con la base de datos en Mysql para relizar un CRUD (create, read, update, delete) y que sea un sistema dinámico y validado.

  • Buscador: Buscador de entradas o contenido en la base de datos y presentado en el sistema.

  • Código: Blog de Videojuegos GitHub.
