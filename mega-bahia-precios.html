Mega Bahía Actualización de Productos

Automatización en el proceso "Actualización de Productos"

Mejor control del cambio de Productos en el sistema local del Almacén, este mini proyecto disminuyó el tiempo en que los empleados se demoraban buscando cada producto nuevo y actualizando en el sistema local uno por uno.


Descripción de la Empresa:

Nombre de la Empresa: Mega Bahía.

Ubicación: Selva Alegre y Gaspar de Carvajal.

Descripción:
Venta de productos tecnológicos y para el hogar.


Aspectos Técnicos:

Herramientas: Java, Netbeans, Programación Orientado a Objetos, librerías para Excel.

Cargo: Desarrollo del Sistema Completo:

Descripción:
El sistema comparaba los productos ya existentes en el sistema local con la nueva mercadería mediante archivos Excel, este arrojaba un tercer archivo Excel que contenia solo los productos que habia que actualizar en precios o crearlos.
Código del proyecto final: Proyecto privado para el Almacén Mega Bahía.
