Venta de Tecnología y productos para el Hogar

Cargo:

Programador y Gestión de Bodega

Período:

Desde: 01-08-2019 Hasta: 15-06-2020

Funciones:
  • Desarrollo de una aplicación en Java para la automatización del proceso "Actualización de precios de porductos de bodega al sistema local"
      • Descripción:
    • Esta aplicación comparaba dos archivos Excel que se realizaban a mano y contenían l información de los productos, este daba como resultado otro Excel que detallaba cuales habian que actualizar de precios y cuales hbía que crear en el sistema Local "Aries" por lo que evitaba largas horas de comparaciones manual
  • Mantenimiento y programación de aplicación web "1000 regalos" para el rápido contacto con los clientes de Mercado libre.
      • Descripción:
    • Se arreglo algunos errores que producia el aplicativo hecho en Codeigniter y se aumento dos modulos par el conteo de clientes y mensajes que contaban cad cuenta en mercado libre. Este aplicativo se conectaba a la api de Mercado libre y Conseguía todos los contactos de clientes que compraban en una sola búsqueda, por lo que ahorraba mucho tiempo en contactos y respuest por parte de la empresa hacia sus clientes.
  • Mantenimiento de Páginas Web Mega Bahía y Multirebajs.
      • Descripción:
    • Actualización y creación de nuevas secciones y páginas de acuerdo a las necesidades del Almacén, estas páginas estaba desarrolladas en Wordpress. Mega Bahía y MultiRebajas
  • Publicación y Actualización de Productos en Mega Bahía.
      • Descripción:
    • Publicación de Productos con SEO y Gestión de Inventario Online en Mega Bahía y MultiRebajas
  • Configuración de un Sistema POS en Mega Bahía.
      • Descripción:
    • Integración de un Sistema POS para la venta online y sincronización de inventario físico con l página web.
