Asesoría Nutricional

Automatización de Negocio

Procesos digitales mediante un sistema y bases de datos localmente, disminución de registros en papel, más control en las sesiones de cad cliente, disponibilidad de horarios, entre otrs ventajas que ofrece la automatización de tus servicios.


Descripción de la Empresa:

Nombre de la Empresa: Asesoría Nutricional.

Ubicación: Hospital Baca Ortiz

Descripción:
Negocio de Asesoría Nutricional, dietas, maquillajes faciales, cuidado nutricional y de salud.


Aspectos Técnicos:

Herramientas: Java, Netbeans, Programación Orientado a Objetos, Git y Github.

Cargo: Módulo Facturas, Agendar Citas y Ventas.

Funciones:

Módulo Ventas y Facturas: Realizar los calculos necesarios para el cobro del servicio, registro de venta y factura en la base de datos.

Módulo Agendar Cita: Registro de una cita que se realiza por llamada telefónica a la base de datos, este desplegaba los horarios disponibles, registraba las sesiones y tratamiento que necesitaban los clientes.

Descripción:
Proyecto relizado en la Universidad para una Asesoría Nutricional, en la que fui el reponsable de la interfaz para Facturas y Ventas, además que fui el responsable de instalar el sistema en el Negocio. Este Sistema utilizaba un conexión a una base de datos MySql localmente.
En mi Modulo me encargaba de realizar las consultas de todos los servicios que disponia en negocio, calculaba precios, iva, descuentos, etc.
Código del proyecto final: Proyecto Asenut
